# cheminformatics_drug_discovery
A hands-on implementation of cheminformatics workflows using RDKit, molecular descriptors, fingerprints, and machine learning models for bioactivity prediction.
# Project Overview

This repository documents my hands-on learning journey in computational drug discovery and cheminformatics using machine learning. The project follows the tutorial series “Drug Discovery using Machine Learning” by Festus, while implementing the concepts through practical coding exercises and experiments using Python.

The goal of this project is to understand how chemical structures can be transformed into numerical representations that allow machine learning models to predict biological activity. Using tools such as RDKit, pandas, scikit-learn, and Jupyter Notebook, the tutorials explore the complete workflow used in early-stage drug discovery.

The repository progresses from fundamental cheminformatics concepts to more advanced topics in quantitative structure–activity relationship (QSAR) modeling and predictive machine learning. Key topics include molecular representation using SMILES, molecular visualization, calculation of physicochemical descriptors, Lipinski’s Rule of Five for drug-likeness evaluation, molecular fingerprints, similarity analysis, and the development of machine learning models for predicting bioactivity.

A major component of the workflow involves working with real-world bioactivity datasets from the ChEMBL database, performing exploratory data analysis, generating molecular fingerprints, and training models such as Random Forest, Support Vector Machines, XGBoost, and K-Nearest Neighbors to predict molecular activity against biological targets.

The later stages of the project focus on model evaluation, feature importance analysis, hyperparameter optimization, and virtual screening of new molecules, demonstrating how computational methods can support medicinal chemists in identifying promising drug candidates.

Overall, this repository serves as a practical guide to modern cheminformatics and machine learning techniques used in drug discovery, providing structured notebooks and code implementations that illustrate the complete pipeline from raw molecular data to predictive models.

# Learning Objectives

Through this project, the following skills are developed:

Understanding SMILES representation of molecules

Visualizing chemical structures using RDKit

Calculating molecular descriptors and Lipinski properties

Generating molecular fingerprints

Measuring molecular similarity using Tanimoto similarity

Performing exploratory data analysis of bioactivity datasets

Building QSAR models using machine learning

Evaluating model performance using cross-validation and ROC analysis

Applying models for virtual screening of new compounds
